Denotation of lexical items
viewed as extensions:
[[triangle]] = the set of triangles

viewed as intensions:
[[triangle]] = 

the characteristic function of the set of triangles, i.e., 
the rule that determines the set of triangles.

this is the same as 
 the property of being a triangle, or 
 the truth function that yields the value 1 for each 

argument that is a triangle, or

 in Fregean terminology: the concept of a triangle.
